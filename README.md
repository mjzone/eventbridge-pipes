## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/my-cdk-app.git


2. **Install dependencies**:
   ```bash
   npm install


3. **Bootstrap CDK**:
   ```bash
   cdk bootstrap


4. **Deploy the stack**:
   ```bash
   cdk deploy
